# 📊 CodeAlpha Data Analytics Internship – Task 2: Exploratory Data Analysis (EDA)

## 📌 Objective
Conducted Exploratory Data Analysis (EDA) on the Superstore Sales dataset to uncover business insights related to sales, profit, discounts, categories, regions, and time-based trends.

## 📂 Dataset
**Superstore Dataset (Kaggle)**  
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- SciPy
- SQL (SQLite)
- Google Colab

## 📈 Key Insights

- Analyzed **9,994 orders** across **4 regions** with **no missing values** or duplicate records, ensuring high-quality data.
- **Technology** generated the highest sales and profit, while **Office Supplies** recorded the highest number of orders.
- Discounts greater than **20%** were strongly associated with negative profit margins.
- A **T-test (p < 0.05)** confirmed that higher discounts significantly reduce profitability.
- The **West** region achieved the highest sales, whereas the **South** region reported the lowest overall profit.
- **Furniture** consistently produced low or negative profits despite strong sales, indicating pricing or cost concerns.
- Correlation analysis showed:
  - **Sales vs Profit:** **0.48** (moderate positive correlation)
  - **Discount vs Profit:** **-0.22** (negative correlation)
- SQL analysis confirmed that **Technology** contributed the highest total profit, while **Furniture** contributed the lowest.
